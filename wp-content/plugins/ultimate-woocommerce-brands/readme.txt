Add Brands taxonomy for products for WooCommerce plugin. Show brand name on product pages and category pages. Use widgets to display brands list.

== Description ==

This plugin allow you to easy add brands to your WooCommerce products on your existing site (with any WordPress theme) and include additional features like:

*   Create unlimited Brands categories for your products (you can assign multiple brands to your one product)
*   Display brand name on Product page (choose from 8 positions to display in Plugin Admin Panel)
*   Display brand name on Category pages and any Product listing pages (Choose from 6 positions to display in Plugin Admin Panel)
*   Filter product by Brands on specials brands pages (every brand on your site will have its own page with assigned products)
*   WordPress Widget to show your brands list in Sidebars in different ways
*   Plugin work with ANY theme that support WooCommerce
*   100% Responsive all plugin elements and pages
*   Translation ready (PO/MO files included)

This is a Free light version of Ultimate WooCommerce Brands Plugin. 
[Purchase PRO version](http://codecanyon.net/item/ultimate-woocommerce-brands-plugin/9433984/?ref=dedalx "PRO version") to get this amazing features:

*   Add logos to your brands
*   Display brand name and/or logo on Product page
*   Special Brands page with all brands listing with ABC filtering (with logos, titles, items counts). Page display style can be changed in various ways in shortcode/VC item settings
*   Special Responsive Brands slider with touch support (you can add it with Shortcode/VC item to any page on your site)
*   Shortcodes for easy usage (Brands listing page, Brands Slider, Products by Brand)
*   WordPress Widget to show your brands list in Sidebars in different ways (logos, titles, counts)
*   WPBakery Visual Composer elements to use instead of shortcodes
*   100% Responsive all plugin elements and pages
*   Free Plugin updates and support
