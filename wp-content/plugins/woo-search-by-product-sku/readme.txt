== Description ==

**Woocommerce Search By SKU allows you to be able to search for prouducts by their sku number. If a full sku is entered then the page is automatically pulled up. It also works to search SKUs in the backend. It does all this while maintaining the normal search functionality.**

### Features

* Install and go. No config necessary at all. Just install and start searching by SKU.

== Installation ==

1. Unzip the download package
1. Upload `tr-wc-search-by-sku` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
